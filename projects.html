Stream.io

The fourth project is a streaming homepage web app inspired by popular platforms like Disney+ and Amazon Prime.

Goal: to delve deeper into Bootstrap components and classes.

I built this project using Bootstrap as a Minimal Viable Product (MVP) to gain a solid understanding of front-end development and layout structuring. The MVP focused on creating a streamlined, functional interface that highlights core elements like navigation, content sections, and responsive design. By leveraging Bootstrap’s component-based structure, I could quickly prototype an adaptable, visually clean layout that captures the essentials. This MVP version reflects the approach companies often require—prioritizing core functionality and usability before moving on to more complex features.

In building this web app, I focused on applying Bootstrap’s components, such as cards for displaying movie thumbnails, a navbar for easy navigation, and carousels to highlight featured content. This project helped me strengthen my understanding of Bootstrap's utility classes and responsive design techniques, giving me a solid foundation for future web design and front-end projects.
